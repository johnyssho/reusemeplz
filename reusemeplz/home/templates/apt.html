Package manager crash course

Správce balíčků (package manager) je nástroj, který usnadňuje instalaci, aktualizaci a odstraňování softwaru na operačním systému Linux. Jeho úlohou je spravovat balíčky, což jsou předpřipravené soubory obsahující aplikace, knihovny a další potřebné soubory.

V Linuxu je jedním z nejčastěji používaných správců balíčků program apt (Advanced Package Tool). Apt je základní nástroj pro správu balíčků v distribucích založených na Debianu, jako je například Ubuntu nebo Debian samotný.

Pamatujte si, že pro použití příkazů apt je obvykle potřeba mít administrátorská práva, která lze získat pomocí příkazu “sudo”.

Takže příkaz pro aktualizaci seznamu balíčků bude ve finále vypadat takto, pokud nebudeme přihlášení jako root.

  sudo apt update

Níže je stručný přehled základních příkazů apt.

  1. Aktualizace seznamu balíčků:
  sudo apt update

Tento příkaz aktualizuje seznam dostupných balíčků z internetových repozitářů.

  1. Aktualizace systému:
  sudo apt upgrade

Tento příkaz aktualizuje všechny nainstalované balíčky na nejnovější verze.

  1. Instalace balíčku:
  sudo apt install nazev_balicku

Tento příkaz nainstaluje vybraný balíček. Nahraďte “nazev_balicku” konkrétním názvem balíčku, který chcete nainstalovat.

  1. Odstranění balíčku:
  sudo apt remove nazev_balicku

Tento příkaz odstraní vybraný balíček ze systému. Nahraďte “nazev_balicku” konkrétním názvem balíčku, který chcete odstranit.

  1. Vyhledání balíčků:
  apt search vyhledavaci_dotaz

Tento příkaz vyhledá balíčky na základě zadaného vyhledávacího dotazu. Nahraďte “vyhledavaci_dotaz” konkrétním výrazem, který chcete vyhledat.

  1. Zobrazení informací o balíčku:
  apt show nazev_balicku

Tento příkaz zobrazí podrobné informace o vybraném balíčku.
